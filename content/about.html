아카이브 소개

이 아카이브는 비지니스 PT 강의에서 배운 내용을 주제별로 구조화하여, 이후 복습·실무 적용·참고 자료로 활용하기 위한 개인 학습 공간입니다.

목적

  • 강의에서 배운 흩어진 내용들을 연결된 구조로 정리하기
  • 핵심 개념·프레임워크·사례를 빠르게 찾아볼 수 있는 검색 가능한 형태로 만들기
  • 시간이 지나도 맥락이 유지되도록 보기 좋은 HTML 문서로 보존하기

폴더 구조

C:\Dev\bizpt\
├── index.html           ← 메인 진입점 (SPA)
├── content\             ← 정리된 강의 HTML 섹션
│   ├── about.html
│   └── lecture-*.html
├── raw\                 ← 원본 자료 (노트, 채팅, 스크린샷 등)
└── assets\
    ├── css\style.css
    └── js\app.js

자료 반영 흐름

  1. raw/ 폴더에 원본 자료를 저장하거나 채팅으로 전달
  2. 내용을 주제별로 분류하고 구조화
  3. content/에 정리된 HTML 섹션으로 변환
  4. assets/js/app.js의 목차(TOC)에 항목 추가

TIP

자료를 전달하실 때는 어떤 강의(회차/주제)의 어떤 부분인지 간단히 함께 알려주시면 분류가 정확해집니다.
