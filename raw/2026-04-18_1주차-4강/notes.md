2. 수익화의 구조

3. 유입 → 수익화 흐름
   - **풀링 콘텐츠**: 유튜브 홈화면에서 *목적 없이 탐색하는* 사람들이 유입되는 콘텐츠.
   - 그렇게 모인 사람들이 **키 콘텐츠**(목적을 가진 콘텐츠)로 이동.
   - 키 콘텐츠는 유튜브 외에도 **네이버 / 유튜브 / 인스타** 같은 다양한 채널에서 만들어진다(목적 있는 검색·탐색 유입).
   - 키 콘텐츠 → **랜딩페이지 접속** → **수익화**.

   ```
   [풀링 콘텐츠]            [키 콘텐츠]              [랜딩페이지]      [수익화]
   유튜브 홈에서            네이버 / 유튜브 /         목적 있는        잠재 → 무료
   목적 없이 탐색  ───►    인스타에서  ───►        진입점     ───►  → 유료
   (탐색 유입)              목적 있는 진입            (전환)           → 재구매
   ```

## [이미지 2] 강의 슬라이드 재구성 (원본 도용 금지 방침 준수)

> 원본 슬라이드 제목: "수익화 구조" (비즈니스 PT 1-4강). 원본 레이아웃·문구를 그대로 옮기지 않고, 강의에서 말한 개념을 내 언어로 재구성.

### 핵심 주장 두 줄 요약
- **기획은 수익화부터 거꾸로 설계**하고, **제작은 풀링 콘텐츠부터 정방향으로** 만든다.
- 이유: *사람이 먼저 모여야 팔린다.* 풀링 콘텐츠가 유입을 만들고, 키 콘텐츠가 목적을 가진 사람을 필터링하고, 그 끝에 수익화가 있다.

### 두 가지 순서 — 방향이 정반대
| 구분 | 단계 1 | 단계 2 | 단계 3 |
|---|---|---|---|
| **기획 순서** (역방향 설계) | ① 수익화 목표 | ② 키 콘텐츠 | ③ 풀링 콘텐츠 |
| **제작 순서** (정방향 실행) | ③ 풀링 콘텐츠 | ② 키 콘텐츠 | ① 수익화 |

즉, 먼저 "뭘로 돈을 벌지"를 정하고 역산해 콘텐츠를 설계하지만, 막상 만들 때는 사람을 모으는 풀링부터 찍는다.

### 채널·의도별 정리
- **풀링 콘텐츠** — 시청자 의도 **없음(탐색형)**. 유튜브 홈 알고리즘에서 노출. 통계적으로 유튜브 유입의 **80% 이상이 홈화면**.
- **키 콘텐츠** — 시청자 의도 **있음(검색형)**. 네이버·인스타·유튜브 검색으로 목적 있는 사람이 들어옴.
- **수익화 단계** — 유입된 사람을 네 단계로 분류: 잠재고객 → 무료고객 → 유료고객 → 재구매고객.

### 제작 파이프라인 (풀링 콘텐츠 제작 실무 순서)
```
썸네일/제목 → 원고 작성 → 촬영 → 편집 → 업로드
```
썸네일·제목을 먼저 확정해야 내용이 그 프레임 안에서 응축된다.

### 한 장짜리 흐름도 (재구성)
```
[사람]
  │  (목적 없이 탐색)
  ▼
풀링 콘텐츠 ──► 키 콘텐츠 ──► 유튜브 본편/랜딩 ──► 수익화
 (유튜브 홈     (네이버/인스타      (전환 지점)         ├ 잠재
  알고리즘)      검색, 목적 O)                         ├ 무료
                                                      ├ 유료
                                                      └ 재구매
```

## [이미지 3] 매출 공식 + 퍼널·전환율 (2026-04-18 전달분, 사용자 구술 요약본)

> 사용자가 이미지 대신 슬라이드 핵심 내용을 요약해 전달. 원본 슬라이드 도용은 없으므로 내 언어로 정리.

### 업로드의 목적
- **성공하는 채널은 업로드하는 목적이 있다.** 단순히 올리는 게 아니라 *왜 올리는가*가 명확해야 함.

### 매출 공식
- **매출 = P (가격) × Q (수량)**
- 예시: **3,000만원 = P(10만원) × Q(10개)**
- 즉, 목표 매출을 먼저 정하면 도달해야 할 Q(판매 수량)가 역산됨 → Q를 만들기 위한 깔때기 입구(조회수)를 설계할 수 있다.

### 깔때기(퍼널) — 전환율 기준 역산
```
풀링 콘텐츠 2,000회
      │ 10%
      ▼
 키 콘텐츠  200회
      │ 10%
      ▼
 랜딩페이지 20명
      │ 50%
      ▼
   구매    10개
```

| 단계 | 수치 | 전환율 |
|---|---|---|
| 풀링 콘텐츠 조회수 | 2,000 | — |
| → 키 콘텐츠 도달 | 200 | **10%** |
| → 랜딩페이지 진입 | 20명 | **10%** |
| → 구매 전환 | 10개 | **50%** |

### 인사이트
- **"랜딩페이지에다가 남기는 %를 봤더니 10%예요"** — 키 콘텐츠 시청자 중 랜딩으로 넘어가는 비율이 약 10%라는 관찰.
- 역산하면 10개를 팔기 위해 풀링 콘텐츠 2,000회 조회가 필요 → **제작 규모·횟수 기획의 근거**.

### 앞선 "기획 역방향" 원칙과의 연결
- 앞서 정리한 "기획은 수익화부터 역산" 원칙이 여기서 **숫자로 구체화**됨.
- 기획 순서: ① 수익화 목표(매출 3천만원) → ② 판매 수량(Q=10) → ③ 랜딩 진입 수(20) → ④ 키 콘텐츠 조회(200) → ⑤ 풀링 콘텐츠 조회(2,000).
