# raw/ 원본 자료 보관소

이 폴더에는 강의 정리에 사용될 **가공 전 원본 자료**를 보관합니다.

## 넣어도 되는 것
- 강의 중 작성한 노트 (`.md`, `.txt`)
- 강의 관련 채팅 로그 / 메신저 캡처
- 강의 슬라이드 / PDF
- 개인 메모, 떠오른 아이디어
- 참고 링크, 관련 블로그 글 저장본

## 권장 파일명 규칙
아무렇게나 넣어도 되지만 아래처럼 작성하면 분류가 쉬워집니다.

```
YYYY-MM-DD_<주제>_<형식>.ext
예) 2026-04-18_포지셔닝_노트.md
    2026-04-18_포지셔닝_채팅로그.txt
    2026-04-18_포지셔닝_슬라이드.pdf
```

폴더로 묶어도 됩니다:
```
raw/
├── 2026-04-18_포지셔닝/
│   ├── note.md
│   ├── chat.txt
│   └── slide.pdf
└── 2026-04-25_타겟팅/
    └── ...
```

## 처리 흐름
1. 이 폴더에 원본 투입 (또는 채팅으로 전달)
2. 주제별로 구조화 → `content/lecture-*.html` 생성
3. `assets/js/app.js`의 TOC에 항목 추가
4. 사이드바에서 바로 열람 가능
